# hollow-feathers
A dark, bird-themed souls-like action RPG built in Unity. Fight through waves of enemies, rest at ancient nests to recover, level up your character, and face a fearsome avian boss in brutal combat. Inspired by the Dark Souls series.

Demo


Tech Stack

Engine: Unity
Language: C#
Version Control: Git/GitHub


Features

Challenging melee combat system with attack, dodge, and defense mechanics
Character leveling and progression system
Health and stamina systems
Nest-based rest points for recovery and respawning
Enemy AI with respawn system
Fearsome bird-themed final boss
Original art, level design, and soundtrack


My Role
I was responsible for the core gameplay systems of the game:

Designed and implemented the full combat system including attacks, hit detection, and player feedback
Built the health and stamina systems governing player and enemy state
Developed the leveling and character progression system
Implemented the nest rest point system for recovery and respawning
Created the enemy spawning and respawn logic
Engineered the core gameplay loop tying all systems together



Getting Started

Clone the repository
Open the project in Unity (version 6000.3.8f1 or higher)
Open the main scene and press Play to run the game
